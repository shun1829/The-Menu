# The-Menu

Goal - その日に使いたい食材を複数入力（例：　鮭、きのこ、ほうれん草、鶏肉、長ネギ、人参、じゃがいも、大根）するとそれを含めた一日3食（主食、主菜、副菜）の献立を表示するアプリを作る。

課程　- クラシルなどの料理アプリに使いたい食材を複数入れると複数全てを使った一品を出力する。
